# send-sms-laravel
send sms in laravel for sms.ir

# Install
`composer require devnull-ir/smssender`

# Use 
add To .env File:
```env
UserApiKey_sms=YourUserApiKey
SecretKey_sms=YourSecretKey
LineNumber_sms=YourPhoneTheSite
```
# Functions

`sendCode()`

`sendMessage()`

`get_token_sms()`
